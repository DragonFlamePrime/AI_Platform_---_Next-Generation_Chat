# ⭐ AI Platform — Next‑Generation Chat + IDE + Memory System

A fully modular, sandboxed, extensible AI platform designed to combine the best features of Copilot, ChatGPT, and Google AI Mode — without their limitations.

This platform supports:

- Unlimited messages  
- Unlimited attachments  
- Branching conversations  
- Linked cross‑chat memory  
- Built‑in sandboxed IDE  
- Image generation  
- Timestamp navigation  
- Import/export conversations  
- Strong safety + jailbreak protection  
- Full sandboxing (no filesystem or network access)  

---

## 🚀 Features

### **Core Chat**
- Unlimited message history  
- Unlimited attachments  
- Smooth navigation  
- Timestamp jump system  
- “Go to Top / Bottom” buttons  

### **Conversation Management**
- Branching conversations  
- Import/export conversations  
- Linked memory groups (user‑controlled)  

### **AI Capabilities**
- Multi‑model routing  
- Attachment‑aware models  
- Image generation  
- Streaming responses  

### **Safety & Sandboxing**
- Sensitive‑data detection  
- Jailbreak prevention  
- Output sanitization  
- Fully sandboxed execution  
- Per‑chat and per‑user isolation  

### **Built‑In IDE**
- Safe code execution  
- No filesystem access  
- No network access  
- Reset after each run  
- Supports Python, JS, and more
